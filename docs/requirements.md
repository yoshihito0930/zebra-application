# スタジオゼブラ アプリケーション要件
## MVP要件
### ユーザ向け
- 予約カレンダーの確認
- 仮予約の申し込み
- 本予約の申し込み
- 予約の変更依頼
- 予約の削除依頼

### 管理者向け
- 予約カレンダーの編集
- 仮予約の承認
- 本予約の承認
- 予約変更の承認
- 予約削除の承認  
  
## 追加要件
### ユーザ向け
- 予約の通知
- 事前決済
- キャンセル待ち機能
- 利用履歴の記録
- 問い合わせ機能
- 多言語対応

### 管理者向け
- アクティブユーザーの確認
- 予約の通知
- 月毎の利用状況の分析
- 問い合わせ・フィードバックの管理
